## bb link

Create local reference symlink to a project or repository.

```
bb link <project-or-repo> [flags]
```

### Options

```
      --absolute         Create absolute symlink instead of relative.
      --as string        Link file/dir name override.
      --catalog string   Catalog override used for auto-clone fallback.
      --dir string       Target directory for the link.
  -h, --help             help for link
```

### Options inherited from parent commands

```

```
