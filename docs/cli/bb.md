## bb

Keep Git repositories consistent across machines.

### Synopsis

bb is a local-first CLI for repository bootstrap and safe cross-machine convergence.

```
bb [flags]
```

### Options

```
  -h, --help    help for bb
  -q, --quiet   Suppress verbose bb logs.
```
